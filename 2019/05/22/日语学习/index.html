  • 日语

    • 发音系统
      • 假名
        • 假借汉字的一部分
        • 平假名
          • 行书和草书的偏旁部首
        • 片假名
          • 楷书的偏旁部首
        • 所以不要说，看着汉字下意识发音
    • 文字系统
      • 假名、汉字、罗马字（官方不可用）
      • 罗马字是一种文字，来源于拉丁文
    • 汉字和假名
      • 一个汉字对应一个假名
      • 日本先把汉字引进去，但是使用的不多，达官贵人才学会。最开始只是表音文字。
      • 选取汉字的行书和楷书的偏旁部首，产生了平假名，男子又产生了片假名
      • 假名发音，先抛弃了汉字，但是发现不行
        • 日语有很多同音异义字，而假名只能发音，不能表义，有可能就会混乱
        • 引入汉字来表意，假名发音，汉字表示含义
        • 单词，用假名书写，然后假名代表这个单词的发音，但是没有含义，然后自己也看不懂，所以配上中文单词代表这个单词的意思。
        • 所以一个音对应的一个日语单词很多，同样对应的汉字就很多，同样就是一个音会对应很多含义，很多中文词。
        • 这样的话，日语单词的含义是由书写中的汉字来决定的，而且就是我们理解的中文的意识！！！喔
